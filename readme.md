ДЗ по тестированию конечного узла https://api.hh.ru/vacancies/
в соответствии с API https://github.com/hhru/api/blob/master/docs/vacancies.md#search

1) Тест дизайн описан и классы эквивалентности описаны в файле test_design.txt
2) Тесты выполнены в виде python-скриптов
3) Краткое описание функционала теста приведено в начале каждого скрипта
4) Каждый тест выдает результат "ОК" или "NOT OK". В случае "NOT OK" в логе теста пишется краткое описание ошибки
5) Для автоматического запуска всех тестов используется скрипт run_tests.py

В последнем тесте обнаружено странное поведение: в документации написано, что при использовании восклицательного знака в поле 
будет искаться точное слово, следующее за "!". Например при запросе "!программурст" должно искаться полное соответствие. 
Однако выдается ненулевое количество вакансий, хотя такого слова в описаниях вакансий нет.
